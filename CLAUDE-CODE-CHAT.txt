● The regex s/#.*// is a sed substitution:

  - s/ — start substitution
  - # — match a literal # (fragment separator in URLs)
  - .* — match any characters after it (zero or more)
  - // — replace the match with nothing (delete it)

  So https://example.com/page#section becomes https://example.com/page.
